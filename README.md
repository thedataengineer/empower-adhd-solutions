# Empower ADHD Solutions

Static recreation of [empoweradhdsolutions.com](https://www.empoweradhdsolutions.com/) for GitHub Pages hosting. Plain HTML/CSS, no build step, no dependencies.

## Pages

- `index.html` — home
- `start-here.html` — three starting paths (recently diagnosed, parents, relationships)
- `coaching.html` — coaching services and pricing
- `blog.html` — blog post index
- `daddhd.html` — DadDHD podcast
- `adhd-info.html` — ADHD education
- `resources.html` — ADHD + autism resource library
- `about.html` — about Braden Young
- `contact.html` — contact and socials
- `privacy.html` — privacy policy

## Local preview

Open `index.html` in a browser, or:

```sh
python3 -m http.server 8000
```

## Deployment

Served by GitHub Pages from the `main` branch root. Push to `main` to deploy.
